Detailed Description

The ImageView class 图像浏览器 

*   由VTK完成相关交互工作
*   支持显示的图像类型有：PNG、JPG、TIF、HRAW
*   按键P移动鼠标可以拾取图像坐标位置和像素值
*   右击弹出右击菜单可以拾取种子坐标点
*   右击菜单可以图像翻转
