Detailed Description

The AngleWidget class. 

*       角度测量器 继承自vtkAngleWidget
*       并封装了vtkTextWidget作为显示面板
*       扩展了point1、center、point2、lineColor、lineWidth、angle等参数接口
*       方便参数设置和读取
*
