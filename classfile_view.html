Detailed Description

The fileView class 文件浏览器 

*        支持文件目录浏览
*        支持重命名，对后缀名更改做警告提示
*        支持键盘上下键翻页
*        地址栏输入地址回车可直接跳转到对应目录
*        提供文件类型筛选器
*
